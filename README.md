# 🌧️ Rainfall Pattern Analysis in India (1901–2015)

A Python data analytics project focused on analyzing long-term rainfall trends, seasonal rainfall contribution, and regional rainfall variability across India using historical datasets.

---

## 📌 Project Overview

India is highly dependent on rainfall for agriculture, water supply, and climate balance.  
This project explores rainfall patterns across Indian subdivisions and districts to understand:

- annual rainfall trends over time
- monsoon dependency
- regional rainfall differences
- rainfall variability across India

The analysis was performed using Python libraries such as Pandas, NumPy, and Matplotlib.

---

## 📂 Dataset Information

### Dataset 1: Rainfall in India (1901–2015)
- File: `rainfall in india 1901-2015.csv`
- Rows: **4116**
- Columns: **19**
- Contains subdivision-wise monthly and yearly rainfall data.

### Dataset 2: District Wise Rainfall Normal
- File: `district wise rainfall normal.csv`
- Rows: **641**
- Columns: **19**
- Contains district-level rainfall normal values.

**Source:** Kaggle / India Meteorological Department (IMD)

---

## 🎯 Study Objectives

- Analyze long-term rainfall trends in India
- Study seasonal rainfall contribution
- Compare rainfall across subdivisions and districts
- Identify rainfall variability and climate inconsistency patterns

---

## ❓ Problem Statement

### Main Problem
To analyze rainfall patterns in India and understand seasonal as well as regional rainfall behavior.

### Sub Problems
- Which regions receive highest and lowest rainfall?
- How has rainfall changed over different decades?
- How dependent is India on monsoon rainfall?

---

## 📊 Analysis Performed

- Data cleaning and missing value handling
- Annual rainfall trend analysis
- Top 10 wettest subdivisions
- Top 10 driest subdivisions
- Seasonal rainfall comparison
- Monthly rainfall pattern analysis
- Rainfall variability analysis
- Decade-wise rainfall trend
- Monsoon vs annual rainfall correlation
- District-wise rainfall comparison

---

## 📈 Key Insights

- Monsoon season contributes the highest share of annual rainfall in India.
- Rainfall distribution is highly uneven across Indian regions.
- Northeastern and coastal regions receive significantly higher rainfall.
- Some subdivisions show high rainfall variability, indicating unpredictable rainfall patterns.
